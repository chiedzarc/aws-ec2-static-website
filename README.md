# Personal Portfolio Website

This is my personal portfolio website built using HTML, CSS, and JavaScript. I deployed it on an AWS EC2 instance using Nginx.

---

(http://51.21.209.229/)

---

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- Nginx (Web Server)
- AWS EC2 (Hosting)
- Git & GitHub (Version Control & Repository)

---

## Deployment Process
1. Developed website locally using HTML, CSS, and JavaScript.
2. Created an AWS EC2 instance (Ubuntu).
3. Installed and configured Nginx web server.
4. Used SCP to transfer files to `/var/www/html/`.
5. Configured security groups to allow HTTP traffic.
6. Accessed live site via public IP address.
